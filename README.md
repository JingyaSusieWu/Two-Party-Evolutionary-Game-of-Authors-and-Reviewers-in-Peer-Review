# Two-Party Evolutionary Game of Authors and Reviewers in Peer Review

This repository provides the MATLAB code and supporting documentation used for the simulation analysis in the manuscript:

**“Incentive Structure and Strategic Evolution in Peer Review: An Analysis Based on Two-Party Evolutionary Game of Authors and Reviewers”**

The repository is intended to support peer review and reproducibility by providing the scripts used to generate the numerical simulation results reported in the manuscript.

## Repository Structure

```text
data_and_code/
├── initial_evolutionary_path/
│   ├── AR_IEP.m
│   └── InitialEvolutionaryPath.m
│
└── parameter_sensitivity/
    ├── AR_PS.m
    ├── ParameterSensitivity_Author.m
    └── ParameterSensitivity_Reviewer.m
```

## Software Requirements

The simulations were conducted using:

* MATLAB R2021b

No additional datasets are required. The scripts use MATLAB’s built-in numerical solver `ode45` and standard plotting functions.

## Overview

This study develops a theoretical two-party evolutionary game model involving authors and reviewers in the peer review process. The MATLAB scripts in this repository are used to simulate the evolutionary dynamics of the system under different initial conditions and parameter settings.

The repository contains two main types of simulation analysis:

1. Initial evolutionary path analysis
2. Parameter sensitivity analysis

## How to Reproduce the Results

To reproduce the simulation results:

1. Download or clone this repository.
2. Open MATLAB R2021b.
3. Navigate to the relevant folder.
4. Run the corresponding main script.

For the initial evolutionary path analysis:

```matlab
cd('data_and_code/initial_evolutionary_path')
InitialEvolutionaryPath
```

For the parameter sensitivity analysis:

```matlab
cd('data_and_code/parameter_sensitivity')
ParameterSensitivity_Author
ParameterSensitivity_Reviewer
```

All figures can be regenerated by running the corresponding MATLAB scripts.

## Reproducibility Notes

The numerical simulations are generated by solving the replicator dynamic equations using MATLAB’s built-in `ode45` solver.

For the initial evolutionary path analysis, the system is simulated from multiple initial conditions over the time interval `[0, 10]`.

For the parameter sensitivity analysis, the system is solved under different parameter settings, and the time evolution of the two state variables is plotted.

The scripts are intended to reproduce the numerical simulation results reported in the manuscript. Users may modify the parameter settings in the scripts to explore additional scenarios.
